# Software Engineering Learning Path

This repository contains code and resources for learning Software Engineering concepts.

## Syllabus

### Software Development Models
- Waterfall Model
- Classic, Iterative, Spiral, Incremental, RAD, Agile, XP, V-Model

### Software Architecture
- System structure
- UI Design, Coupling/Cohesion

### Software Project Management
- Planning, Organizing, Controlling
- Cost Estimation (COCOMO), CMM, Risk Management

### Software Metrics, Requirements, Configuration, Quality, Design, Reliability, Testing, and Maintenance.
